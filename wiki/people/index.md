# People

One page per person in your orbit.

| Name | Created | Status |
|------|---------|--------|
| (empty — run the wizard to seed 3-5 people stubs + `me.md`) | | |

## How this works

- Wizard seeds `me.md` + 3-5 people stubs based on Q3 of the first-run setup
- `/ingest` auto-creates new pages when sources mention people who don't have one yet
- `/absorb` writes provenance into existing pages as sources accumulate

See [`rules/wiki-conventions.md`](../../rules/wiki-conventions.md).
